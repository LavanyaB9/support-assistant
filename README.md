 Support Assistant (Streamlit App)

A simple and smart FAQ-based Support Assistant built using Python and Streamlit 
This assistant automatically answers common user queries and escalates complex questions to a human agent.


 Features

 Instant FAQ lookup  
 Keyword-based query detection  
 Auto-escalation for unknown or complex queries  
 Clean chat-style UI using Streamlit  
 Easily expandable FAQ database  
Stores chat history during the session  


 Tech Stack
Python 3.8+
Streamlit
Dictionary-based FAQ engine

 Installation

1. Clone the repository:

git clone https://github.com/LavanyaB9/support-assistant.git
cd support-assistant
python -m pip install streamlit
python -m streamlit run app.py
